# Movie Scraper API

Sistema de scraping multi-proveedor para peliculas, series, animes y doramas

## Características

- 13 proveedores de contenido activos
- Busqueda multi-proveedor en paralelo
- Extraccion de servidores de video
- Integracion con TMDB
- Frontend estilo Netflix
- CLI interactivo

## Proveedores

- estrenoscinesaa
- verpelistv
- sololatino
- pelisplushd
- detodopeliculas
- cinecalidad
- serieskao

## Instalacion

1. git clone https://github.com/SpAy2024/movie-scraper-api.git
2. cd movie-scraper-api
3. npm install
4. npm run dev
